üöÄ Getting Started

Welcome to ETF Dashboard!

Follow these steps to get started with analyzing ETFs:

  1. Explore the Dashboard: The main dashboard shows an overview of all ETFs with their current RSI values
  2. Filter by RSI: Use the threshold input to filter ETFs below a certain RSI level (default: 30 for oversold)
  3. Analyze Individual ETFs: Click on any ETF to view detailed charts and analysis
  4. Set Up Your Portfolio: Add your holdings in the Portfolio section to track performance
  5. Create Alerts: Set up notifications for price or RSI changes
Pro Tip: Start by looking at the oversold ETFs (RSI < 30) as they may present buying opportunities!

üìà Understanding RSI

What is RSI?

The Relative Strength Index (RSI) is a momentum oscillator that measures the speed and change of price movements. It ranges from 0 to 100.

RSI Interpretation
RSI < 30

Oversold
Potential Buy Signal

RSI 30-70

Normal Range
Hold Signal

RSI > 70

Overbought
Potential Sell Signal

How RSI is Calculated

RSI = 100 - (100 / (1 + RS))

Where RS = Average Gain / Average Loss over a specified period (typically 14 days)

Important: RSI is a tool, not a guarantee. Always consider other factors and use proper risk management.

üîî Setting Up Alerts

Creating Smart Alerts
Available Alert Types
  • Price Above: Alert when price exceeds a value
  • Price Below: Alert when price drops below a value
  • RSI Above: Alert when RSI rises above a level
  • RSI Below: Alert when RSI falls below a level
  • Volume Above: Alert when volume exceeds threshold
Best Practices
  • Set RSI alerts at key levels (30 for oversold, 70 for overbought)
  • Use price alerts for support/resistance levels
  • Enable email notifications for important alerts
  • Regularly review and update your alerts
  • Don't set too many alerts to avoid notification fatigue
Tip: Start with RSI alerts below 30 to catch potential buying opportunities!
